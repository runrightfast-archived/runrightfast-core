STS 3.4.0 - based on Eclipse Kepler 4.3.1
=========================================
Install the following STS extensions:
- Language and Frameworks
  - Grails (current production)
- Language and Frameowrk Tooling
  - Gradle Support
  - Grails Support
  - Groovy 2.1 Compiler for Groovy-Eclipse
  - Groovy-Eclipse
- Utilities
  - EclEmma
  - FindBugs
  - PMD
  
Install the following Eclipse Plugins from the Eclipse Marketplace:
- Markdown Editor
- CodeNarc Eclipse
  
Gradle Project Import Instructions
==================================
After cloning the GIT repositories, perform a "Gradle Project" import into the workspace
